# 外贸英语教练

这是一个本地可用的外贸英语学习小程序原型，覆盖听、说、读、写四类练习，并内置客户消息助手。

## 使用方式

直接打开 `index.html` 即可使用。推荐使用 Chrome 或 Edge，因为语音朗读和跟读识别对浏览器支持有要求。

## 已包含功能

- 6 个外贸场景课程：询盘、报价、样品、订单付款、物流、售后。
- 每课包含听力、口语、阅读、写作练习。
- 客户消息助手支持粘贴英文或中文消息，做场景翻译、意图识别、关键词提取和英文回复建议。
- 支持原文朗读、译文朗读、英文回复朗读。
- 支持本地学习进度保存。
